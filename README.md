# Виджет банковских операций клиента

## Суть проекта:

IT-отделу крупного банка поручено добавить новую функциональность для личного кабинета клиента.
Нужно разработать виджет, который показывает несколько последних успешных банковских операций клиента.
Этот проект является реализацией бэкенд составляющей. По завершению разработки, программа будет готовить
данные для отображения в новом виджете на стороне фронтэнда.

### Технический стек:

Для проекта требуется версия python не ниже `3.11`  
А так же версия Poetry не ниже `2.1.4`

В проекте используются:

* Менеджер пакетов `Poetry`.
* Линтеры: `mypy`, `black`, `isort`, `flake8`
* Библиотека `datetime`
* Фреймворк управления тестами `pytest`
* Инструмент сбора статистики покрытия кода `pytest-cov`

### Тесты и тестирование:

#### Аннотация 
Тесты расположены в пакете "tests" в корне проекта. Тесты параметризованы с помощью маркировки
`pytest.mark.parametrise()`. Тесты гораздо легче использовать благодаря тому, что в качестве значений параметров в
маркировку передаются не фиксированные значения, в результаты вычисления двух функций-форматеров `param_packer` и
`value_packer` в которые передается набор тестовых данных, приводимый ими к нужному для маркировки состоянию. Что
позволяет менять не сами тесты, а только сами наборы данных.
Наборы данных вынесены в отдельный пакет и разбиты тематически соответственно названию тестируемого модуля и функции.

#### Запуск тестов
Все тесты разом можно запустить командой:

```bash
  pytest
  ```
Если требуется запустить конкретные тесты отдельно, то используйте команды:

* Запуск конкретного теста:
```bash
  pytest -k 'Имя теста или его часть'
  ```
Часть имени теста должна достаточно четко указывать на этот самый тест. Чтобы в прогон не попали другие, обладающие сходным наименованием с общей частью в названии.

* Запуск тестов со сбором покрытия:

```bash
  pytest --cov=src --cov-report=html
  ```
Перед запуском команды для полной уверенности в актуальности данных отслеживания покрытия рекомендуется удалить из проекта пакет htmlcov и файл .coverage.
